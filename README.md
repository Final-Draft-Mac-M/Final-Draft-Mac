## Overview

Final Draft Professional Suite is a complete screenwriting platform with production tools, reporting, and multi-format export for professional script development.

Production tools generate scene reports, character reports, cast lists, and shooting schedules from the script content, bridging pre-production planning with the screenplay.

Multi-format export produces scripts in PDF, FDX, Fountain, and other formats for distribution to different production departments.

Night mode and customizable page display adapt the writing environment to personal preferences for extended writing sessions.

Final Draft Professional Suite is the screenwriting platform for professional writers who need industry-standard formatting with the production tools and collaboration that professional script development demands.

## Performance & Integration

Final Draft is engineered around macOS technologies that handle performance-critical work efficiently. Tasks run in the background without slowing down the rest of your system.

The app integrates deeply with the macOS ecosystem out of the box:

- Launch workflows directly through **Spotlight**
- Preview files instantly with **Quick Look**
- Trigger actions from any app via the **Services menu**
- Fully compatible with **Shortcuts** for automated workflows
